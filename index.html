巻Resources

Everything this site is built from — no build tools, no frameworks, just HTML, CSS and a bit of JavaScript. Listed here so future-me (or anyone remixing this) knows exactly what's under the hood.

Typography

Headings use Shippori Mincho, a Japanese serif; body text uses Noto Sans JP. Both loaded free from Google Fonts.

Icons

All social icons (Facebook, Instagram, TikTok, GitHub) and the page markers are hand-coded inline SVG — no icon library or external requests.

Structure

Semantic HTML5 with plain CSS3 (custom properties, clip-path, grid, scroll-snap) and a small vanilla JavaScript file — no React, no build step.

Artwork

The banner and page panels use original CSS gradients rather than official posters or screenshots, to keep the template copyright-clean. Swap in your own licensed images or fan art before publishing.

Interactions

The banner slideshow, active-link highlighting, and scroll-snap navigation are all handled by the inline <script> at the bottom of the file — under 60 lines.

Hosting

Being a single static file, it can be dropped straight into GitHub Pages, Netlify, or Vercel with zero configuration.
